Network and Transport Layers

Theoretical Part

  • TCP Protocol: Description of its functionality and features.
  • UDP Protocol: Overview and comparison with TCP.
  • TCP Parameters: Sequence numbers, acknowledgments, etc.
  • UDP Parameters: Packet size, checksum, etc.
  • TCP Connection: Three-way handshake, connection establishment, and release.
  • Ports: Explanation of local and remote ports.

Practical Part

  1. Accessing Google: Use netstat and Wireshark to identify ports for www.google.com.
  2. File Transfer with FTP: Transfer a file and analyze ports using netstat and Wireshark.
  3. File Transfer with SFTP/SCP: Transfer a file and analyze protocol and port usage.
  4. DNS Protocol: Use Wireshark to identify the transport protocol and port used by DNS.
